Create SD card block device with SPI.

Create a block device object for an SPI-connected SD or MMC card.

Parameters
spi_instSPI instance, as defined in the pico-sdk hardware_spi library
mosiSPI Master Out Slave In(TX) pin
misoSPI Master In Slave Out(RX) pin
scklSPI clock pin
csSPI Chip select pin
hzSPI clock frequency (Hz)
enable_crcBoolean value to enable CRC on read/write
Returns
Block device object. Returnes NULL in case of failure.
Return values
NULLFailed to create block device object.
