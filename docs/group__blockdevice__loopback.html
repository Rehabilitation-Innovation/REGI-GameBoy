blockdevice_t * blockdevice_loopback_create ( const char * path,
size_t capacity,
size_t block_size )

Create loopback block device.

Create a loopback device object that uses a disk image file. Specify the file path allocated to the block device, as well as the maximum size capacity and block size block_size.

Parameters
pathDisk image file path.
capacityMaximum device size bytes.
block_sizeBlock size byte.
Returns
Block device object. Returnes NULL in case of failure.
Return values
NULLFailed to create block device object.
