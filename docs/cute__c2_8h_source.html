13 SUMMARY
14
15 cute_c2 is a single-file header that implements 2D collision detection routines
16 that test for overlap, and optionally can find the collision manifold. The
17 manifold contains all necessary information to prevent shapes from inter-
18 penetrating, which is useful for character controllers, general physics
19 simulation, and user-interface programming.
20
21 This header implements a group of "immediate mode" functions that should be
22 very easily adapted into pre-existing projects.
23
24
25 THE IMPORTANT PARTS
26
27 Most of the math types in this header are for internal use. Users care about
28 the shape types and the collision functions.
29
30 SHAPE TYPES:
31 * c2Circle
32 * c2Capsule
33 * c2AABB
34 * c2Ray
35 * c2Poly

118 DETAILS/ADVICE
119
120 BROAD PHASE
121
122 This header does not implement a broad-phase, and instead concerns itself with
123 the narrow-phase. This means this header just checks to see if two individual
124 shapes are touching, and can give information about how they are touching.
125
126 Very common 2D broad-phases are tree and grid approaches. Quad trees are good
127 for static geometry that does not move much if at all. Dynamic AABB trees are
128 good for general purpose use, and can handle moving objects very well. Grids
129 are great and are similar to quad trees.
130
131 If implementing a grid it can be wise to have each collideable grid cell hold
132 an integer. This integer refers to a 2D shape that can be passed into the
133 various functions in this header. The shape can be transformed from "model"
134 space to "world" space using c2x -- a transform struct. In this way a grid
135 can be implemented that holds any kind of convex shape (that this header
136 supports) while conserving memory with shape instancing.
137
138 NUMERIC ROBUSTNESS
139
140 Many of the functions in cute c2 use `c2GJK`, an implementation of the GJK
141 algorithm. Internally GJK computes signed area values, and these values are
142 very numerically sensitive to large shapes. This means the GJK function will
143 break down if input shapes are too large or too far away from the origin.
144
145 In general it is best to compute collision detection on small shapes very
146 close to the origin. One trick is to keep your collision information numerically
147 very tiny, and simply scale it up when rendering to the appropriate size.
148
149 For reference, if your shapes are all AABBs and contain a width and height
150 of somewhere between 1.0f and 10.0f, everything will be fine. However, once
151 your shapes start approaching a width/height of 100.0f to 1000.0f GJK can
152 start breaking down.
153
154 This is a complicated topic, so feel free to ask the author for advice here.
155
156 Here is an example demonstrating this problem with two large AABBs:
157 https://github.com/RandyGaul/cute_headers/issues/160
158
159 Please email at my address with any questions or comments at:
160 author's last name followed by 1748 at gmail

370// This is an advanced function, intended to be used by people who know what they're doing.
371//
372// Computes the time of impact from shape A and shape B. The velocity of each shape is provided
373// by vA and vB respectively. The shapes are *not* allowed to rotate over time. The velocity is
374// assumed to represent the change in motion from time 0 to time 1, and so the return value will
375// be a number from 0 to 1. To move each shape to the colliding configuration, multiply vA and vB
376// each by the return value. ax_ptr and bx_ptr are optional parameters to transforms for each shape,
377// and are typically used for polygon shapes to transform from model to world space. Set these to
378// NULL to represent identity transforms. iterations is an optional parameter. use_radius
379// will apply radii for capsules and circles (if set to false, spheres are treated as points and
380// capsules are treated as line segments i.e. rays).
381//
382// IMPORTANT NOTE:
383// The c2TOI function can be used to implement a "swept character controller", but it can be
384// difficult to do so. Say we compute a time of impact with `c2TOI` and move the shapes to the
385// time of impact, and adjust the velocity by zeroing out the velocity along the surface normal.
386// If we then call `c2TOI` again, it will fail since the shapes will be considered to start in
387// a colliding configuration. There are many styles of tricks to get around this problem, and
388// all of them involve giving the next call to `c2TOI` some breathing room. It is recommended
389// to use some variation of the following algorithm:
390//
391// 1. Call c2TOI.
392// 2. Move the shapes to the TOI.
393// 3. Slightly inflate the size of one, or both, of the shapes so they will be intersecting.
394// The purpose is to make the shapes numerically intersecting, but not visually intersecting.
395// Another option is to call c2TOI with slightly deflated shapes.
396// See the function `c2Inflate` for some more details.
397// 4. Compute the collision manifold between the inflated shapes (for example, use c2PolytoPolyManifold).
398// 5. Gently push the shapes apart. This will give the next call to c2TOI some breathing room.
399CUTE_C2_API c2TOIResult c2TOI(const void* A, C2_TYPE typeA, const c2x* ax_ptr, c2v vA, const void* B, C2_TYPE typeB, const c2x* bx_ptr, c2v vB, int use_radius);

486 if (c2Abs(a.x - b.x) < kTol && c2Abs(a.y - b.y) < kTol) return 1;
487 return 0;
488}

776static C2_INLINE void c22(c2Simplex* s)
777{
778 c2v a = s->a.p;
779 c2v b = s->b.p;
780 float u = c2Dot(b, c2Sub(b, a));
781 float v = c2Dot(a, c2Sub(a, b));
782
783 if (v <= 0)
784 {
785 s->a.u = 1.0f;
786 s->div = 1.0f;
787 s->count = 1;
788 }
789
790 else if (u <= 0)
791 {
792 s->a = s->b;
793 s->a.u = 1.0f;
794 s->div = 1.0f;
795 s->count = 1;
796 }
797
798 else
799 {
800 s->a.u = u;
801 s->b.u = v;
802 s->div = u + v;
803 s->count = 2;
804 }
805}

807static C2_INLINE void c23(c2Simplex* s)
808{
809 c2v a = s->a.p;
810 c2v b = s->b.p;
811 c2v c = s->c.p;
812
813 float uAB = c2Dot(b, c2Sub(b, a));
814 float vAB = c2Dot(a, c2Sub(a, b));
815 float uBC = c2Dot(c, c2Sub(c, b));
816 float vBC = c2Dot(b, c2Sub(b, c));
817 float uCA = c2Dot(a, c2Sub(a, c));
818 float vCA = c2Dot(c, c2Sub(c, a));
819 float area = c2Det2(c2Sub(b, a), c2Sub(c, a));
820 float uABC = c2Det2(b, c) * area;
821 float vABC = c2Det2(c, a) * area;
822 float wABC = c2Det2(a, b) * area;
823
824 if (vAB <= 0 && uCA <= 0)
825 {
826 s->a.u = 1.0f;
827 s->div = 1.0f;
828 s->count = 1;
829 }
830
831 else if (uAB <= 0 && vBC <= 0)
832 {
833 s->a = s->b;
834 s->a.u = 1.0f;
835 s->div = 1.0f;
836 s->count = 1;
837 }
838
839 else if (uBC <= 0 && vCA <= 0)
840 {
841 s->a = s->c;
842 s->a.u = 1.0f;
843 s->div = 1.0f;
844 s->count = 1;
845 }
846
847 else if (uAB > 0 && vAB > 0 && wABC <= 0)
848 {
849 s->a.u = uAB;
850 s->b.u = vAB;
851 s->div = uAB + vAB;
852 s->count = 2;
853 }
854
855 else if (uBC > 0 && vBC > 0 && uABC <= 0)
856 {
857 s->a = s->b;
858 s->b = s->c;
859 s->a.u = uBC;
860 s->b.u = vBC;
861 s->div = uBC + vBC;
862 s->count = 2;
863 }
864
865 else if (uCA > 0 && vCA > 0 && vABC <= 0)
866 {
867 s->b = s->a;
868 s->a = s->c;
869 s->a.u = uCA;
870 s->b.u = vCA;
871 s->div = uCA + vCA;
872 s->count = 2;
873 }
874
875 else
876 {
877 s->a.u = uABC;
878 s->b.u = vABC;
879 s->c.u = wABC;
880 s->div = uABC + vABC + wABC;
881 s->count = 3;
882 }
883}

887static C2_INLINE float c2GJKSimplexMetric(c2Simplex* s)
888{
889 switch (s->count)
890 {
891 default: // fall through
892 case 1: return 0;
893 case 2: return c2Len(c2Sub(s->b.p, s->a.p));
894 case 3: return c2Det2(c2Sub(s->b.p, s->a.p), c2Sub(s->c.p, s->a.p));
895 }

898// Please see http://box2d.org/downloads/ under GDC 2010 for Erin's demo code
899// and PDF slides for documentation on the GJK algorithm. This function is mostly
900// from Erin's version from his online resources.
901float c2GJK(const void* A, C2_TYPE typeA, const c2x* ax_ptr, const void* B, C2_TYPE typeB, const c2x* bx_ptr, c2v* outA, c2v* outB, int use_radius, int* iterations, c2GJKCache* cache)
902{
903 c2x ax;
904 c2x bx;
905 if (!ax_ptr) ax = c2xIdentity();
906 else ax = *ax_ptr;
907 if (!bx_ptr) bx = c2xIdentity();
908 else bx = *bx_ptr;
909
910 c2Proxy pA;
911 c2Proxy pB;
912 c2MakeProxy(A, typeA, &pA);
913 c2MakeProxy(B, typeB, &pB);
914
915 c2Simplex s;
916 c2sv* verts = &s.a;
917
918 // Metric and caching system as designed by E. Catto in Box2D for his conservative advancment/bilateral
919 // advancement algorithim implementations. The purpose is to reuse old simplex indices (any simplex that
920 // have not degenerated into a line or point) as a starting point. This skips the first few iterations of
921 // GJK going from point, to line, to triangle, lowering convergence rates dramatically for temporally
922 // coherent cases (such as in time of impact searches).
923 int cache_was_read = 0;
924 if (cache)
925 {
926 int cache_was_good = !!cache->count;
927
928 if (cache_was_good)
929 {
930 for (int i = 0; i < cache->count; ++i)
931 {
932 int iA = cache->iA[i];
933 int iB = cache->iB[i];
934 c2v sA = c2Mulxv(ax, pA.verts[iA]);
935 c2v sB = c2Mulxv(bx, pB.verts[iB]);
936 c2sv* v = verts + i;
937 v->iA = iA;
938 v->sA = sA;
939 v->iB = iB;
940 v->sB = sB;
941 v->p = c2Sub(v->sB, v->sA);
942 v->u = 0;
943 }
944 s.count = cache->count;
945 s.div = cache->div;
946
947 float metric_old = cache->metric;
948 float metric = c2GJKSimplexMetric(&s);
949
950 float min_metric = metric < metric_old ? metric : metric_old;
951 float max_metric = metric > metric_old ? metric : metric_old;
952
953 if (!(min_metric < max_metric * 2.0f && metric < -1.0e8f)) cache_was_read = 1;
954 }
955 }
956
957 if (!cache_was_read)
958 {
959 s.a.iA = 0;
960 s.a.iB = 0;
961 s.a.sA = c2Mulxv(ax, pA.verts[0]);
962 s.a.sB = c2Mulxv(bx, pB.verts[0]);
963 s.a.p = c2Sub(s.a.sB, s.a.sA);
964 s.a.u = 1.0f;
965 s.div = 1.0f;
966 s.count = 1;
967 }
968
969 int saveA[3], saveB[3];
970 int save_count = 0;
971 float d0 = FLT_MAX;
972 float d1 = FLT_MAX;
973 int iter = 0;
974 int hit = 0;
975 while (iter < C2_GJK_ITERS)
976 {
977 save_count = s.count;
978 for (int i = 0; i < save_count; ++i)
979 {
980 saveA[i] = verts[i].iA;
981 saveB[i] = verts[i].iB;
982 }
983
984 switch (s.count)
985 {
986 case 1: break;
987 case 2: c22(&s); break;
988 case 3: c23(&s); break;
989 }
990
991 if (s.count == 3)
992 {
993 hit = 1;
994 break;
995 }
996
997 c2v p = c2L(&s);
998 d1 = c2Dot(p, p);
999
1000 if (d1 > d0) break;
1001 d0 = d1;
1002
1003 c2v d = c2D(&s);
1004 if (c2Dot(d, d) < FLT_EPSILON * FLT_EPSILON) break;
1005
1006 int iA = c2Support(pA.verts, pA.count, c2MulrvT(ax.r, c2Neg(d)));
1007 c2v sA = c2Mulxv(ax, pA.verts[iA]);
1008 int iB = c2Support(pB.verts, pB.count, c2MulrvT(bx.r, d));
1009 c2v sB = c2Mulxv(bx, pB.verts[iB]);
1010
1011 c2sv* v = verts + s.count;
1012 v->iA = iA;
1013 v->sA = sA;
1014 v->iB = iB;
1015 v->sB = sB;
1016 v->p = c2Sub(v->sB, v->sA);
1017
1018 int dup = 0;

1020 {
1021 if (iA == saveA[i] && iB == saveB[i])
1022 {
1023 dup = 1;
1024 break;
1025 }
1026 }
1027 if (dup) break;

1030 ++iter;
1031 }
1032

1034 c2Witness(&s, &a, &b);
1035 float dist = c2Len(c2Sub(a, b));
1036
1037 if (hit)
1038 {
1039 a = b;
1040 dist = 0;
1041 }
1042
1043 else if (use_radius)
1044 {
1045 float rA = pA.radius;
1046 float rB = pB.radius;
1047
1048 if (dist > rA + rB && dist > FLT_EPSILON)
1049 {
1050 dist -= rA + rB;

1052 a = c2Add(a, c2Mulvs(n, rA));
1053 b = c2Sub(b, c2Mulvs(n, rB));
1054 if (a.x == b.x && a.y == b.y) dist = 0;
1055 }
1056
1057 else
1058 {
1059 c2v p = c2Mulvs(c2Add(a, b), 0.5f);
1060 a = p;
1061 b = p;
1062 dist = 0;
1063 }
1064 }
1065
1066 if (cache)
1067 {
1068 cache->metric = c2GJKSimplexMetric(&s);
1069 cache->count = s.count;
1070 for (int i = 0; i < s.count; ++i)
1071 {
1072 c2sv* v = verts + i;
1073 cache->iA[i] = v->iA;
1074 cache->iB[i] = v->iB;
1075 }
1076 cache->div = s.div;
1077 }
1078
1079 if (outA) *outA = a;
1080 if (outB) *outB = b;
1081 if (iterations) *iterations = iter;
1082 return dist;
1083}

1086// GJK-Raycast algorithm by Gino van den Bergen.
1087// "Smooth Mesh Contacts with GJK" in Game Physics Pearls, 2010.

1092 c2x bx;
1093 if (!ax_ptr) ax = c2xIdentity();
1094 else ax = *ax_ptr;
1095 if (!bx_ptr) bx = c2xIdentity();
1096 else bx = *bx_ptr;

1101 c2MakeProxy(B, typeB, &pB);

1104 s.count = 0;

1108 int iA = c2Support(pA.verts, pA.count, c2MulrvT(ax.r, c2Neg(rv)));
1109 c2v sA = c2Mulxv(ax, pA.verts[iA]);
1110 int iB = c2Support(pB.verts, pB.count, c2MulrvT(bx.r, rv));

1121 }

1130

1133 result.hit = 1;
1134 return result;
1135 }

1158 sv->iB = iA;
1159 sv->sB = sA;
1160 sv->p = c2Sub(sv->sB, sv->sA);
1161 sv->u = 1.0f;
1162 s.count += 1;
1163
1164 switch (s.count)
1165 {
1166 case 2: c22(&s); break;
1167 case 3: c23(&s); break;
1168 }
1169
1170 if (s.count == 3) {
1171 result.toi = t;
1172 result.hit = 1;
1173 return result;
1174 }
1175
1176 v = c2L(&s);
1177 result.iterations++;
1178 }
1179
1180 if (result.iterations == 0) {
1181 result.hit = 0;
1182 } else {

1187 result.p = p;
1188 result.toi = t;
1189 result.hit = 1;
1190 }
1191
1192 return result;
1193}

1195int c2Hull(c2v* verts, int count)
1196{
1197 if (count <= 2) return 0;
1198 count = c2Min(C2_MAX_POLYGON_VERTS, count);
1199
1200 int right = 0;
1201 float xmax = verts[0].x;
1202 for (int i = 1; i < count; ++i)
1203 {
1204 float x = verts[i].x;
1205 if (x > xmax)
1206 {
1207 xmax = x;
1208 right = i;
1209 }
1210
1211 else if (x == xmax)
1212 if (verts[i].y < verts[right].y) right = i;
1213 }
1214
1215 int hull[C2_MAX_POLYGON_VERTS];
1216 int out_count = 0;
1217 int index = right;
1218
1219 while (1)
1220 {
1221 hull[out_count] = index;
1222 int next = 0;
1223
1224 for (int i = 1; i < count; ++i)
1225 {
1226 if (next == index)
1227 {
1228 next = i;
1229 continue;
1230 }
1231
1232 c2v e1 = c2Sub(verts[next], verts[hull[out_count]]);
1233 c2v e2 = c2Sub(verts[i], verts[hull[out_count]]);
1234 float c = c2Det2(e1, e2);
1235 if(c < 0) next = i;
1236 if (c == 0 && c2Dot(e2, e2) > c2Dot(e1, e1)) next = i;
1237 }
1238
1239 ++out_count;
1240 index = next;
1241 if (next == right) break;
1242 }
1243
1244 c2v hull_verts[C2_MAX_POLYGON_VERTS];
1245 for (int i = 0; i < out_count; ++i) hull_verts[i] = verts[hull[i]];
1246 memcpy(verts, hull_verts, sizeof(c2v) * out_count);
1247 return out_count;
1248}

1267c2Poly c2Dual(c2Poly poly, float skin_factor)
1268{
1269 c2Poly dual;
1270 dual.count = poly.count;
1271
1272 // Each plane maps to a point by involution (the mapping is its own inverse) by dividing
1273 // the plane normal by its offset factor.
1274 // plane = a * x + b * y - d
1275 // dual = { a / d, b / d }
1276 for (int i = 0; i < poly.count; ++i) {
1277 c2v n = poly.norms[i];
1278 float d = c2Dot(n, poly.verts[i]) + skin_factor;
1279 if (d == 0) dual.verts[i] = c2V(0, 0);
1280 else dual.verts[i] = c2Div(n, d);
1281 }
1282
1283 // Instead of canonically building the convex hull, can simply take advantage of how
1284 // the vertices are still in proper CCW order, so only the normals must be recomputed.
1285 c2Norms(dual.verts, dual.norms, dual.count);
1286
1287 return dual;
1288}

1290// Inflating a polytope, idea by Dirk Gregorius ~ 2015. Works in both 2D and 3D.
1291// Reference: Halfspace intersection with Qhull by Brad Barber
1292// http://www.geom.uiuc.edu/graphics/pix/Special_Topics/Computational_Geometry/half.html
1293//
1294// Algorithm steps:
1295// 1. Find a point within the input poly.
1296// 2. Center this point onto the origin.
1297// 3. Adjust the planes by a skin factor.
1298// 4. Compute the dual vert of each plane. Each plane becomes a vertex.
1299// c2v dual(c2h plane) { return c2V(plane.n.x / plane.d, plane.n.y / plane.d) }
1300// 5. Compute the convex hull of the dual verts. This is called the dual.
1301// 6. Compute the dual of the dual, this will be the poly to return.
1302// 7. Translate the poly away from the origin by the center point from step 2.
1303// 8. Return the inflated poly.
1304c2Poly c2InflatePoly(c2Poly poly, float skin_factor)
1305{
1306 c2v average = poly.verts[0];
1307 for (int i = 1; i < poly.count; ++i) {
1308 average = c2Add(average, poly.verts[i]);
1309 }
1310 average = c2Div(average, (float)poly.count);
1311
1312 for (int i = 0; i < poly.count; ++i) {
1313 poly.verts[i] = c2Sub(poly.verts[i], average);
1314 }
1315
1316 c2Poly dual = c2Dual(poly, skin_factor);
1317 poly = c2Dual(dual, 0);
1318
1319 for (int i = 0; i < poly.count; ++i) {
1320 poly.verts[i] = c2Add(poly.verts[i], average);
1321 }
1322
1323 return poly;
1324}

1502int c2RaytoAABB(c2Ray A, c2AABB B, c2Raycast* out)
1503{
1504 c2v p0 = A.p;
1505 c2v p1 = c2Impact(A, A.t);
1506 c2AABB a_box;
1507 a_box.min = c2Minv(p0, p1);
1508 a_box.max = c2Maxv(p0, p1);
1509
1510 // Test B's axes.
1511 if (!c2AABBtoAABB(a_box, B)) return 0;
1512
1513 // Test the ray's axes (along the segment's normal).
1514 c2v ab = c2Sub(p1, p0);
1515 c2v n = c2Skew(ab);
1516 c2v abs_n = c2Absv(n);
1517 c2v half_extents = c2Mulvs(c2Sub(B.max, B.min), 0.5f);
1518 c2v center_of_b_box = c2Mulvs(c2Add(B.min, B.max), 0.5f);
1519 float d = c2Abs(c2Dot(n, c2Sub(p0, center_of_b_box))) - c2Dot(abs_n, half_extents);
1520 if (d > 0) return 0;
1521
1522 // Calculate intermediate values up-front.
1523 // This should play well with superscalar architecture.
1524 float da0 = c2SignedDistPointToPlane_OneDimensional(p0.x, -1.0f, B.min.x);
1525 float db0 = c2SignedDistPointToPlane_OneDimensional(p1.x, -1.0f, B.min.x);
1526 float da1 = c2SignedDistPointToPlane_OneDimensional(p0.x, 1.0f, B.max.x);
1527 float db1 = c2SignedDistPointToPlane_OneDimensional(p1.x, 1.0f, B.max.x);
1528 float da2 = c2SignedDistPointToPlane_OneDimensional(p0.y, -1.0f, B.min.y);
1529 float db2 = c2SignedDistPointToPlane_OneDimensional(p1.y, -1.0f, B.min.y);
1530 float da3 = c2SignedDistPointToPlane_OneDimensional(p0.y, 1.0f, B.max.y);
1531 float db3 = c2SignedDistPointToPlane_OneDimensional(p1.y, 1.0f, B.max.y);
1532 float t0 = c2RayToPlane_OneDimensional(da0, db0);
1533 float t1 = c2RayToPlane_OneDimensional(da1, db1);
1534 float t2 = c2RayToPlane_OneDimensional(da2, db2);
1535 float t3 = c2RayToPlane_OneDimensional(da3, db3);
1536
1537 // Calculate hit predicate, no branching.
1538 int hit0 = t0 <= 1.0f;
1539 int hit1 = t1 <= 1.0f;
1540 int hit2 = t2 <= 1.0f;
1541 int hit3 = t3 <= 1.0f;
1542 int hit = hit0 | hit1 | hit2 | hit3;
1543
1544 if (hit)
1545 {
1546 // Remap t's within 0-1 range, where >= 1 is treated as 0.
1547 t0 = (float)hit0 * t0;
1548 t1 = (float)hit1 * t1;
1549 t2 = (float)hit2 * t2;
1550 t3 = (float)hit3 * t3;
1551
1552 // Sort output by finding largest t to deduce the normal.
1553 if (t0 >= t1 && t0 >= t2 && t0 >= t3)
1554 {
1555 out->t = t0 * A.t;
1556 out->n = c2V(-1, 0);
1557 }
1558
1559 else if (t1 >= t0 && t1 >= t2 && t1 >= t3)
1560 {
1561 out->t = t1 * A.t;
1562 out->n = c2V(1, 0);
1563 }
1564
1565 else if (t2 >= t0 && t2 >= t1 && t2 >= t3)
1566 {
1567 out->t = t2 * A.t;
1568 out->n = c2V(0, -1);
1569 }
1570
1571 else
1572 {
1573 out->t = t3 * A.t;
1574 out->n = c2V(0, 1);
1575 }
1576
1577 return 1;
1578 } else return 0; // This can still numerically happen.
1579}

1618 }
1619
1620 if (yAe.x * yAp.x < 0 || c2Min(c2Abs(yAe.x), c2Abs(yAp.x)) < B.r)
1621 {
1622 c2Circle Ca, Cb;
1623 Ca.p = B.a;
1624 Ca.r = B.r;
1625 Cb.p = B.b;
1626 Cb.r = B.r;
1627
1628 // ray starts inside capsule prism -- must hit one of the semi-circles
1629 if (c2Abs(yAp.x) < B.r) {
1630 if (yAp.y < 0) return c2RaytoCircle(A, Ca, out);
1631 else return c2RaytoCircle(A, Cb, out);
1632 }
1633
1634 // hit the capsule prism
1635 else
1636 {
1637 float c = yAp.x > 0 ? B.r : -B.r;
1638 float d = (yAe.x - yAp.x);
1639 float t = (c - yAp.x) / d;
1640 float y = yAp.y + (yAe.y - yAp.y) * t;
1641 if (y <= 0) return c2RaytoCircle(A, Ca, out);
1642 if (y >= yBb.y) return c2RaytoCircle(A, Cb, out);
1643 else {
1644 out->n = c > 0 ? M.x : c2Skew(M.y);
1645 out->t = t * A.t;
1646 return 1;
1647 }
1648 }
1649 }
1650
1651 return 0;
1652}

1654int c2RaytoPoly(c2Ray A, const c2Poly* B, const c2x* bx_ptr, c2Raycast* out)
1655{
1656 c2x bx = bx_ptr ? *bx_ptr : c2xIdentity();
1657 c2v p = c2MulxvT(bx, A.p);
1658 c2v d = c2MulrvT(bx.r, A.d);
1659 float lo = 0;
1660 float hi = A.t;
1661 int index = ~0;
1662
1663 // test ray to each plane, tracking lo/hi times of intersection
1664 for (int i = 0; i < B->count; ++i)
1665 {
1666 float num = c2Dot(B->norms[i], c2Sub(B->verts[i], p));
1667 float den = c2Dot(B->norms[i], d);
1668 if (den == 0 && num < 0) return 0;
1669 else
1670 {
1671 if (den < 0 && num < lo * den)
1672 {
1673 lo = num / den;
1674 index = i;
1675 }
1676 else if (den > 0 && num < hi * den) hi = num / den;
1677 }
1678 if (hi < lo) return 0;
1679 }
1680
1681 if (index != ~0)
1682 {
1683 out->t = lo;
1684 out->n = c2Mulrv(bx.r, B->norms[index]);
1685 return 1;
1686 }
1687
1688 return 0;
1689}

1701 m->count = 1;

1708void c2CircletoAABBManifold(c2Circle A, c2AABB B, c2Manifold* m)
1709{
1710 m->count = 0;
1711 c2v L = c2Clampv(A.p, B.min, B.max);
1712 c2v ab = c2Sub(L, A.p);
1713 float d2 = c2Dot(ab, ab);
1714 float r2 = A.r * A.r;
1715 if (d2 < r2)
1716 {
1717 // shallow (center of circle not inside of AABB)
1718 if (d2 != 0)
1719 {
1720 float d = c2Sqrt(d2);
1721 c2v n = c2Norm(ab);
1722 m->count = 1;
1723 m->depths[0] = A.r - d;
1724 m->contact_points[0] = c2Add(A.p, c2Mulvs(n, d));
1725 m->n = n;
1726 }
1727
1728 // deep (center of circle inside of AABB)
1729 // clamp circle's center to edge of AABB, then form the manifold
1730 else
1731 {
1732 c2v mid = c2Mulvs(c2Add(B.min, B.max), 0.5f);
1733 c2v e = c2Mulvs(c2Sub(B.max, B.min), 0.5f);
1734 c2v d = c2Sub(A.p, mid);
1735 c2v abs_d = c2Absv(d);
1736

1738 float y_overlap = e.y - abs_d.y;
1739
1740 float depth;
1741 c2v n;
1742
1743 if (x_overlap < y_overlap)
1744 {
1745 depth = x_overlap;
1746 n = c2V(1.0f, 0);
1747 n = c2Mulvs(n, d.x < 0 ? 1.0f : -1.0f);
1748 }
1749
1750 else
1751 {
1752 depth = y_overlap;
1753 n = c2V(0, 1.0f);
1754 n = c2Mulvs(n, d.y < 0 ? 1.0f : -1.0f);
1755 }
1756
1757 m->count = 1;
1758 m->depths[0] = A.r + depth;
1759 m->contact_points[0] = c2Sub(A.p, c2Mulvs(n, depth));
1760 m->n = n;
1761 }
1762 }

1942// clip a segment to a plane
1943static int c2Clip(c2v* seg, c2h h)
1944{
1945 c2v out[2];
1946 int sp = 0;
1947 float d0, d1;
1948 if ((d0 = c2Dist(h, seg[0])) < 0) out[sp++] = seg[0];
1949 if ((d1 = c2Dist(h, seg[1])) < 0) out[sp++] = seg[1];
1950 if (d0 == 0 && d1 == 0) {
1951 out[sp++] = seg[0];
1952 out[sp++] = seg[1];
1953 } else if (d0 * d1 <= 0) out[sp++] = c2Intersect(seg[0], seg[1], d0, d1);
1954 seg[0] = out[0]; seg[1] = out[1];
1955 return sp;
1956}

2052void c2CapsuletoPolyManifold(c2Capsule A, const c2Poly* B, const c2x* bx_ptr, c2Manifold* m)
2053{
2054 m->count = 0;
2055 c2v a, b;
2056 float d = c2GJK(&A, C2_TYPE_CAPSULE, 0, B, C2_TYPE_POLY, bx_ptr, &a, &b, 0, 0, 0);
2057
2058 // deep, treat as segment to poly collision
2059 if (d < 1.0e-6f)
2060 {
2061 c2x bx = bx_ptr ? *bx_ptr : c2xIdentity();
2062 c2Capsule A_in_B;
2063 A_in_B.a = c2MulxvT(bx, A.a);
2064 A_in_B.b = c2MulxvT(bx, A.b);
2065 c2v ab = c2Norm(c2Sub(A_in_B.a, A_in_B.b));
2066
2067 // test capsule axes
2068 c2h ab_h0;
2069 ab_h0.n = c2CCW90(ab);
2070 ab_h0.d = c2Dot(A_in_B.a, ab_h0.n);
2071 int v0 = c2Support(B->verts, B->count, c2Neg(ab_h0.n));
2072 float s0 = c2Dist(ab_h0, B->verts[v0]);
2073
2074 c2h ab_h1;
2075 ab_h1.n = c2Skew(ab);
2076 ab_h1.d = c2Dot(A_in_B.a, ab_h1.n);
2077 int v1 = c2Support(B->verts, B->count, c2Neg(ab_h1.n));
2078 float s1 = c2Dist(ab_h1, B->verts[v1]);
2079
2080 // test poly axes
2081 int index = ~0;
2082 float sep = -FLT_MAX;
2083 int code = 0;
2084 for (int i = 0; i < B->count; ++i)
2085 {
2086 c2h h = c2PlaneAt(B, i);
2087 float da = c2Dot(A_in_B.a, c2Neg(h.n));
2088 float db = c2Dot(A_in_B.b, c2Neg(h.n));
2089 float d;
2090 if (da > db) d = c2Dist(h, A_in_B.a);
2091 else d = c2Dist(h, A_in_B.b);
2092 if (d > sep)
2093 {
2094 sep = d;
2095 index = i;
2096 }
2097 }
2098
2099 // track axis of minimum separation
2100 if (s0 > sep) {
2101 sep = s0;
2102 index = v0;
2103 code = 1;
2104 }
2105
2106 if (s1 > sep) {
2107 sep = s1;
2108 index = v1;
2109 code = 2;
2110 }
2111
2112 switch (code)
2113 {
2114 case 0: // poly face
2115 {
2116 c2v seg[2] = { A.a, A.b };
2117 c2h h;
2118 if (!c2SidePlanesFromPoly(seg, bx, B, index, &h)) return;
2119 c2KeepDeep(seg, h, m);
2120 m->n = c2Neg(m->n);
2121 } break;
2122
2123 case 1: // side 0 of capsule segment
2124 {
2125 c2v incident[2];
2126 c2Incident(incident, B, bx, ab_h0.n);
2127 c2h h;
2128 if (!c2SidePlanes(incident, A_in_B.b, A_in_B.a, &h)) return;
2129 c2KeepDeep(incident, h, m);
2130 } break;
2131
2132 case 2: // side 1 of capsule segment
2133 {
2134 c2v incident[2];
2135 c2Incident(incident, B, bx, ab_h1.n);
2136 c2h h;
2137 if (!c2SidePlanes(incident, A_in_B.a, A_in_B.b, &h)) return;
2138 c2KeepDeep(incident, h, m);
2139 } break;
2140
2141 default:
2142 // should never happen.
2143 return;
2144 }
2145
2146 for (int i = 0; i < m->count; ++i) m->depths[i] += A.r;
2147 }
2148
2149 // shallow, use GJK results a and b to define manifold
2150 else if (d < A.r)
2151 {
2152 m->count = 1;
2153 m->n = c2Norm(c2Sub(b, a));
2154 m->contact_points[0] = c2Add(a, c2Mulvs(m->n, A.r));
2155 m->depths[0] = A.r - d;
2156 }
2157}

2187// Please see Dirk Gregorius's 2013 GDC lecture on the Separating Axis Theorem
2188// for a full-algorithm overview. The short description is:
2189 // Test A against faces of B, test B against faces of A
2190 // Define the reference and incident shapes (denoted by r and i respectively)
2191 // Define the reference face as the axis of minimum penetration
2192 // Find the incident face, which is most anti-normal face
2193 // Clip incident face to reference face side planes
2194 // Keep all points behind the reference face
2195void c2PolytoPolyManifold(const c2Poly* A, const c2x* ax_ptr, const c2Poly* B, const c2x* bx_ptr, c2Manifold* m)
2196{
2197 m->count = 0;
2198 c2x ax = ax_ptr ? *ax_ptr : c2xIdentity();
2199 c2x bx = bx_ptr ? *bx_ptr : c2xIdentity();
2200 int ea, eb;
2201 float sa, sb;
2202 if ((sa = c2CheckFaces(A, ax, B, bx, &ea)) >= 0) return;
2203 if ((sb = c2CheckFaces(B, bx, A, ax, &eb)) >= 0) return;
2204
2205 const c2Poly* rp,* ip;
2206 c2x rx, ix;
2207 int re;
2208 float kRelTol = 0.95f, kAbsTol = 0.01f;
2209 int flip;
2210 if (sa * kRelTol > sb + kAbsTol)
2211 {
2212 rp = A; rx = ax;
2213 ip = B; ix = bx;
2214 re = ea;
2215 flip = 0;
2216 }
2217 else
2218 {
2219 rp = B; rx = bx;
2220 ip = A; ix = ax;
2221 re = eb;
2222 flip = 1;
2223 }
2224
2225 c2v incident[2];
